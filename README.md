## Requirements

- iOS 16.0+
- Xcode 15.0+
- Swift 6.0

## Setup

1. Open the project in Xcode
2. Build and run on simulator or device
3. Select a photo and discover your aesthetic style!

## How It Works

1. User selects or takes a photo
2. On-device analysis using Apple's Vision Framework and Core ML
3. Body structure, proportions, and colors are analyzed
4. Aesthetic profile scores are calculated
5. Top 3 results are displayed with personalized commentary

## Notes

- Uses on-device machine learning for all analysis
- All processing happens on background thread
- Results include color statistics, body analysis, and style commentary
- Premium subscription available for advanced features
